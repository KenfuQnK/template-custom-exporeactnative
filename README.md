# Plantilla Expo React Native (SDK 54) + NativeWind (Tailwind)

Esta es una plantilla base personalizada para el desarrollo de aplicaciones móviles con **Expo SDK 54**, diseñada para ser rápida, moderna y fácil de escalar.

## 🚀 Características Principales

- **Expo SDK 54**: La versión más reciente y estable de Expo.
- **NativeWind (v4)**: Implementación de Tailwind CSS para React Native que utiliza estilos nativos.
- **Expo Router**: Enrutamiento basado en archivos (File-based routing), similar a Next.js.
- **TypeScript**: Configuración completa para un desarrollo tipado y seguro.
- **React Navigation v7**: Navegación nativa fluida.
- **Reanimated & Gesture Handler**: Listos para crear animaciones y gestos complejos.

## 🛠️ Herramientas de Utilidad Incluidas

- **Prettier + ESLint**: Configuración robusta para mantener el código limpio.
- **Prettier Tailwind Plugin**: Ordena automáticamente tus clases de Tailwind para mayor legibilidad.
- **Vector Icons**: Acceso inmediato a miles de iconos con `@expo/vector-icons`.
- **Estructura de Carpetas Limpia**: Separación clara entre `app`, `components` y `constants`.
- **Scripts de Calidad**:
  - `npm run lint`: Revisa errores de estilo y código.
  - `npm run format`: Corrige automáticamente el formato del código.

## ⚙️ Configuración Inicial

1. **Instalar dependencias**:
   
   ```bash
   npm install
   ```

2. **Iniciar el proyecto**:
   
   ```bash
   npm run start
   ```
   
   Para emular el proyecto en un dispositivo físico mediante Expo Go (útil si hay problemas de red local):
   
   ```bash
   npm run start:tunnel
   ```

3. **Plataformas específicas**:
   
   - `npm run android`
   - `npm run ios`
   - `npm run web`

## 📁 Estructura del Proyecto

- `/app`: Rutas de la aplicación (Expo Router).
- `/components`: Componentes reutilizables de UI.
- `/constants`: Colores, temas y configuraciones globales.
- `/assets`: Imágenes, fuentes y recursos estáticos.
